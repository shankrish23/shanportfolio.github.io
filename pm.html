Project Management

In the "Project Management" subject, I gained a comprehensive understanding of essential project management techniques and tools to effectively plan, execute, and control projects.
Through various assignments and a group project, I explored financial analysis, work breakdown structure, critical path analysis, earned value analysis, and network diagrams to manage and monitor project progress.






Assignment

Financial Analysis

I analyzed projects using financial metrics such as the discounted payback period, net present value (NPV), profitability index (PI), payback period, and internal rate of return (IRR). These techniques allowed me to assess project feasibility and make financially sound decisions.


Work Breakdown Structure (WBS) Bridge

I created a detailed WBS bridge, breaking down complex projects into manageable tasks and sub-tasks. The WBS facilitated resource allocation, cost estimation, and efficient project execution.


Critical Path Analysis using QM Software

Using Quantitative Methods (QM) software, I performed critical path analysis to identify the most critical activities in a project, helping to schedule tasks effectively and minimize project duration.


Balance Sheet and Earned Value Analysis

I utilized balance sheets to assess a project's financial performance and identify potential financial risks. I also conducted earned value analysis to measure project progress and performance against planned costs and schedules.


Critical Ratio and S-Curve

I calculated the critical ratio to prioritize project activities based on their impact on project duration. I also created S-curves to visualize project progress over time and monitor performance.


A-O-N Network Diagram

I constructed Activity-on-Node (A-O-N) network diagrams to depict task dependencies and facilitate critical path analysis.
